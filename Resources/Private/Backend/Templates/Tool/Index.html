  • Check Media storage index

    Detect whether a file is existing in the database but missing in the storage. The tool can also detect duplicate file objects in the database. Check index

    The same can be achieved with a CLI command:

    cd {sitePath};
./typo3/cli_dispatch.phpsh extbase media:checkIndex

  • Index Media storage

    Indexing of all files within the Media storage can be done by a scheduler task or by a CLI command:

    cd {sitePath};
./typo3/cli_dispatch.phpsh extbase media:index
